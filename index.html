💻 What you need
A web browser (Chrome or Edge), access to python.microbit.org/v/3, and optionally a BBC micro:bit. The built-in simulator means you can do everything online — no hardware required!
🧠 What you'll learn
Computational thinking (decomposition, patterns, abstraction, algorithms), Python programming, and how to design, build and debug your own micro:bit projects.

Each lesson has a thinking activity, a theory section, a coding tutorial, a practice task, and prompt to update a personal progress log. You build skills lesson by lesson.

The Four Computational Thinking Pillars

🧩 Decomposition
Breaking a big problem into smaller, manageable pieces
🔍 Pattern Recognition
Spotting similarities, repetition and patterns in problems
🎭 Abstraction
Focusing on what matters and ignoring unnecessary details
📝 Algorithm Design
Creating step-by-step instructions to solve any problem

Level Up YourUnderstanding in this course

Uni-Structural Level
You focus on one isolated concept. You can write a single command or define a term, but you cannot connect it to other code or use it to solve problems
Multi-Structural Level
You handle multiple coding components but treat them as separate pieces. You write linear steps but struggle to integrate them or see how they interact within a full program.
Relational Level
You can see connections between different concepts and understand how they relate to each other. You understand how variables, loops, and functions connect, allowing you to build, explain, and debug working, structured software systems.
Multi-Relational Level
You can integrate multiple concepts and see how they work together to solve complex problems. You look beyond the immediate program to optimize algorithm efficiency, build highly reusable frameworks, or apply computational logic to non-coding, real-world problems.
